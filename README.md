Project #1: ATB - All Things Basketball. (HTML, CSS, SASS, Bootstrap)

This first project was a great introduction in creating a website using HTML for the skeleton, CSS & SASS for styling, and utilizing many Bootstrap 5.3 components throughout the project.
The theme of the project is ATB - All Things Basketball. The goal was to create a website for casual and enthusiast basketball fans alike, providing them with an all around
great space to learn statistics, follow their favourite teams, read news and watch highlights.

Features included in the website:

- Pop-up cookie modal
- Collapsable navbar with logo
- Self scrolling carousel with controls and indicators for basketball related news
- Infinitely scrolling banner that displays sponsors
- MVPs of the Week: three player display for the best players of the past week with their statistics
- Scoreboards for the three respective leagues along with the date of the match, team logos, names and scores
- Latest basketball news that can be accessed using links featured in the site
- Video highlights: Autplayed three featured video highlights with controls of the leagues featured.
- Button that displays a modal to subscribe to the sites newsletter present along with a form that has validation
- Footer with site description and animations for links and socials

Project #2: Recipe Manager (JavaScript, jquery, AJAX)

This second project was an excellent opportunity to dive deeper into web development by utilizing JavaScript for dynamic content, AJAX for seamless data interaction, and jQuery for simplified DOM manipulation.
The theme of the project is a Recipe Manager. The goal was to create a robust web application that allows users to manage their favorite recipes efficiently. This project offers a comprehensive solution for food enthusiasts, providing a user-friendly interface to add, edit, and delete recipes. Each recipe card displays essential information, including ingredients, instructions, and an appetizing image. By leveraging AJAX, the application ensures smooth and asynchronous data updates without refreshing the page, enhancing the overall user experience.

Features included in the website:

- User Sign-up and form validation
- Single Page Application (SPAPP) integration
- Toggleable gallery with modal pop ups
- Data driven content loading
- Add, edit or delete functions for each of the recipes
- FAQ section with accordion menu
- Theme switching (light mode and dark mode)
- Floating nav menu
